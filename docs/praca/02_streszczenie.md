# Streszczenie

Praca dotyczy projektu aplikacji wspierającej wyszukiwanie rejsów żeglarskich oraz kojarzenie załóg i organizatorów. Celem było stworzenie rozwiązania, które oprócz przeglądania ofert ułatwia budowanie zaufania między użytkownikami poprzez profil, historię aktywności i wspólne kontakty.
